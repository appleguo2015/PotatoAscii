# Potato Ascii
## A Simple Tool By aPpLegUo
It just can show ascii in terminal.

## How To Install
bash download.sh at [here](
    https://github.com/appleguo2015/PotatoAscii/blob/master/download.sh
) and bash it:
``` bash
./download.sh
```

## How To Use
Run it with a file argument:
```bash
paf <file>
```

Examples:
```bash
paf apple.txt
paf potato.txt
paf aPpLegUo.txt
```
