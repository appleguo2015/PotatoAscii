# Potato Ascii
## A Simple Tool By aPpLegUo
it just can show ascii in terminal
## How To Use
1. download it from [here](
    https://github.com/appleguo2015/PotatoAscii/releases
)
2. run it
```bash
./paf <file>
```
u can run it like this
```bash
./paf apple.txt
./paf potato.txt
./paf aPpLegUo.txt
```
